# iqp-boost

Minimal experiments for IQP ensemble boosting on binary datasets (BAS, parity, Gaussian mixture, and shapes).

## Run Experiments

```bash
uv run main.py
uv run main_parity.py
uv run main_gaussian.py
uv run main_shapes.py
```

## Outputs

Runs write logs and plots under `out/` (timestamped run folders).
